## 二、出发前准备

### 2.1 签证

**✅ 免签！无需提前办理。**

泰国对中国公民实施免签政策（2025年有效），停留30天内免签。持有效中国护照直接入境。

> ⚠️ 出发前请在**外交部官网**确认最新政策，以防临时变动。

**入境所需材料清单：**
- [ ] 护照（有效期须距出行日期 **6个月以上**）
- [ ] 回程机票订单截图（入境时可能被查验）
- [ ] 酒店预订确认单（截图备用）

### 2.2 旅行保险

强烈建议购买，推荐**众安"全球无忧"**或**携程旅行险**（与机票打包购买更便宜）。

| 保障项目 | 建议保额 |
|---------|---------|
| 境外医疗 | ≥ 50万元 |
| 航班延误/取消 | ≥ 500元/次 |
| 行李丢失/损坏 | ≥ 3000元 |
| 演唱会取消险 | 按票价投保 |

> 💰 参考价格：**约80元/人**（3天综合险），2人合计约 **160元**。

### 2.3 通信

**推荐：** 抵达廊曼机场后在 **AIS 专柜**购买旅游SIM卡。
- 3天不限流量套餐：**约100泰铢/张（约21元）**
- 含语音通话，信号全国覆盖

### 2.4 货币

- 建议出发前在国内银行换取 **3000泰铢/人**（约638元）现金，用于市场、小摊等不能刷卡的场合
- 大商场、酒店、连锁餐厅均可刷 **Visa/MasterCard**
- 机场ATM有手续费，尽量少用

### 2.5 行前打包清单

**证件类：**
- [ ] 护照（✅ 最重要）
- [ ] 机票订单 + 酒店订单（电子截图）
- [ ] 旅行保险单

**衣物类（5月曼谷气候：32~35°C，高湿热，午后阵雨）：**
- [ ] 轻薄短袖、短裤（3~4套）
- [ ] 薄外套（室内冷气超强，演唱会场馆更冷）
- [ ] 泳衣（Day 2 珊瑚岛必备）
- [ ] 防晒帽、太阳镜
- [ ] 凉鞋 + 运动鞋各一双
- [ ] 演唱会服装（应援色/偶像同款）

**日用品：**
- [ ] 防晒霜 SPF50+（每2小时补涂）
- [ ] 防蚊液（5月雨季蚊虫多）
- [ ] 肠胃药（黄连素/诺氟沙星）
- [ ] 充电宝 10000mAh+
- [ ] 折叠雨伞（午后阵雨频繁）
- [ ] 防水袋（珊瑚岛玩水用）

**演唱会专用：**
